# Discover Karnataka

## Overview

**Discover Karnataka** is a responsive tourism website that showcases the rich culture, heritage, historical landmarks, and natural beauty of Karnataka. The project is designed to provide visitors with an engaging platform to explore the state's traditions, tourist attractions, cultural events, and important destinations through an interactive web interface.

## Features

* Responsive and user-friendly website design
* Home page with an introduction to Karnataka
* Dedicated page highlighting Karnataka's culture and traditions
* Tourist attractions and gallery section
* Embedded multimedia content for an engaging experience
* Contact page for visitor inquiries
* Custom 404 error page
* Clean navigation with an attractive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Responsive Web Design

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/praveenhallur123/Discover-Karnataka.git
   ```

2. Navigate to the project folder:

   ```bash
   cd Discover-Karnataka
   ```

3. Open `index.html` in any modern web browser.

## Website Sections

* **Home:** Introduction to Karnataka.
* **Culture:** Information about Karnataka's traditions, festivals, language, cuisine, and heritage.
* **Gallery:** Images showcasing famous tourist destinations and scenic locations.
* **Report:** Additional project information.
* **Contact:** Contact form for user interaction.
